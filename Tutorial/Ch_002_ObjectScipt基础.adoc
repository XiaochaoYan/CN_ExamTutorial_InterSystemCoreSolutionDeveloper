=== 3.ObjectScript 的基本SQL操作
可以直接使用SQL语句

---- 
Query Select() As %SQLQuery [SqlProc]
{
  SELECT %ID, FirstName, LastName 
  FROM Person
  ORDER By LastName, FirstName
}
---- 
以上代码为声明一个SQL查询用的method
